# Быстрый старт — Gemma Windows 2026

1. **Двойной клик** по **`gemma-windows-2026.exe`** в корне репозитория  
   или запустите **`START.bat`**
2. Либо **[Releases](../../releases/latest)** → скачайте тот же **`gemma-windows-2026.exe`**
3. Windows может спросить SmartScreen — «Подробнее» → «Выполнить»

> Файл **`gemma-windows-2026.exe`** — установщик/лаунчер под это репозиторий (gemma-windows-2026).
